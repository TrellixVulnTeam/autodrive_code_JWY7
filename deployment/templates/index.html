Methodology:

This model is used for segment the road from the background in the aeriel images. Furthermore, it supports the clients to find the shortest path between two points that was chosen by the clients We collected 9532 aeriel satellite images of the New York City as the dataset and used the NYC roadmap to label the data for this Deep Learning model. Our model is trained based on DeepLab V3+ model and its pre-trained model on Cityscapes Semantic Segmentation Dataset. Minimal adjustment was done on the initial architecture because this is our first project in Deep Learning. The resulted model achieves a satisfying mIoU accuracy of 0.905 (background), 0.768 (road), and 0.836 (overall). Then, we used the Skimage library to find the shortest path between two chosen points. Finally, we deployed this model on a simple website hosted on Google Cloud Platform. The frontend and backend of this website is kept minimal because our main focus in this project is the Deep Learning model itself. You can view the video demo here: Link
